■ GitHub에 올리는 방법 (한 번만 하면 됨)
   저장소: https://github.com/YURIIEUN/260311

1. Git 설치
   https://git-scm.com/download/win
   → 다운로드 후 설치 (기본 설정 그대로 Next)

2. 설치가 끝나면 컴퓨터를 한 번 재시작하거나,
   Cursor/터미널을 완전히 닫았다가 다시 켜기.

3. 이 폴더에서 "push.bat" 파일을 더블클릭.

4. 처음 실행이면:
   - 저장소 초기화 후 자동으로 GitHub(origin) 연결
   - 커밋 메시지 물어보면 Enter만 누르면 됨 (기본: 로또 번호 추천 서비스 업데이트)
   - push 완료될 때까지 기다리기

5. GitHub 로그인 창이 뜨면:
   - 브라우저에서 로그인하거나
   - Personal Access Token 사용 (설정에서 생성 가능)

이후에는 수정할 때마다 push.bat만 더블클릭하면 자동으로 반영됩니다.
